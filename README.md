## Project Overview

The Snapdeal Database Management System is designed to manage the core operations of an e-commerce platform using a relational database. The system stores and organizes information related to customers, sellers, products, categories, orders, payments, deliveries, and product reviews. By using SQL, the database ensures accurate data storage, minimizes redundancy, and enables fast data retrieval. The project demonstrates how a well-structured database supports efficient online shopping operations and simplifies business data management.

## Features

* Customer registration and profile management.
* Seller registration and product management.
* Product catalog with categories and pricing.
* Order placement and order tracking.
* Payment management, including online payment and Cash on Delivery (COD).
* Delivery status tracking.
* Inventory and stock management.
* Product ratings and customer reviews.
* Fast data retrieval using SQL queries.
* Secure and organized data storage with reduced redundancy.
* Report generation for customers, products, orders, and sales analysis.
